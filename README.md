# Proton ETF 1 – BFMC 2026 Reports Repository

## Overview
This repository contains official **Bosch Future Mobility Challenge (BFMC) 2026** reports submitted by **Team Proton ETF 1**.  
The purpose of this repository is to serve as a centralized place for uploading and archiving project reports required by BFMC organizers, as well as preserving documentation for future generations of competitors.

The project focuses on the development of **autonomous driving algorithms for a 1/10 scale vehicle**, including perception, decision-making, and control.

## Report Submission Timeline
The following deadlines are defined by the BFMC competition rules:

- **1st Report – December 22, 2025**  
  The team must demonstrate manual control of the vehicle using the provided Bosch start-up code.

- **2nd Report – February 6, 2026**  
  The team must establish a basic data flow from sensors to actuators (e.g., camera input linked to steering control).

- **Qualifications – March 9, 2026**  
  The vehicle should demonstrate basic autonomous behavior, including lane keeping and simple reactions to standard obstacles (e.g., stopping at a stop sign).

- **3rd Report – April 13, 2026**  
  The vehicle should demonstrate nearly complete autonomous functionality, including handling complex intersections and reacting appropriately to obstacles.
